# Nuvix Desktop

Este app desktop abre a versão online do Nuvix em uma janela própria.

URL carregada:

```text
https://nuvix-xi.vercel.app
```

## Ícone

Para o instalador Windows usar o ícone oficial, coloque na raiz do projeto:

```text
icone.ico
```

Depois gere o instalador:

```bash
npm run desktop:build
```

O instalador será gerado em `dist-desktop/`.
