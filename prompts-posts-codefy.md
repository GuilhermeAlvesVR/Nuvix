# Prompts para posts do Instagram — Codefy

## Post 1 — Apresentação (quem somos)

```
Crie um post de apresentação para o Instagram da Codefy, uma fábrica de software sob medida. A Codefy cria sistemas personalizados para qualquer tipo de negócio que precisa organizar processos, clientes, agenda ou finanças.

Tom: profissional, direto, confiável. Público: donos de pequenas e médias empresas no Brasil que ainda usam planilha, papel ou sistemas genéricos que não atendem 100% as necessidades deles.

Formato: legenda para post no feed (cerca de 200-300 caracteres). Inclua 3-5 hashtags. A primeira frase precisa prender a atenção.

Ideia central: apresentar a Codefy como a empresa que transforma a gestão do negócio com tecnologia feita sob medida, sem complicação.
```

## Post 2 — O problema (software genérico vs sob medida)

```
Crie um post para Instagram da Codefy sobre o problema de usar software genérico que não se adapta ao seu negócio.

Tom: direto, identificação com a dor do empresário. Público: donos de empresa que já compraram sistema pronto mas tiveram que se adaptar a ele, em vez do sistema se adaptar a eles.

Legenda: 200-300 caracteres. Comece com uma pergunta ou afirmação forte. Ex: "Já comprou um sistema pronto e descobriu que ele não faz o que sua empresa precisa?".

Hashtags: 3-5. Inclua um CTA para o link da bio.
```

## Post 3 — O que a Codefy faz (serviço)

```
Crie um post para Instagram explicando o que a Codefy faz: cria sistemas sob medida para empresas.

Tom: explicativo, objetivo, destacando benefícios práticos. Público: empresários que querem resolver problemas específicos de gestão.

Legenda: 200-300 caracteres.

Explique que a Codefy:
- Escuta o processo do cliente
- Cria um sistema feito para a realidade da empresa
- Entrega pronto para usar, sem assinatura mensal alta ou funções inúteis

Feche com um CTA: link na bio para contato.

Hashtags: #softwarepersonalizado #fabricaessoftware #codefy #sistemasobmedida
```

## Post 4 — Exemplos de nichos

```
Crie um post para Instagram da Codefy mostrando que ela atende qualquer nicho. Exemplos: clínicas, oficinas, escolas, escritórios de advocacia, corretoras, academias, etc.

Legenda: curta, 150-200 caracteres. Lista rápida de 4-5 exemplos de negócios que podem ter um sistema sob medida. Inclua hashtags e CTA para link da bio.
```


## Post 6 — Recap (carrossel)

