# Prompts para criar logo da Codefy no Gemini

## Opção 1 — Ícone + texto (horizontal)

```
Crie um logotipo horizontal para "Codefy", uma empresa de software para clínicas e consultórios. Estilo minimalista premium, cores escuras (preto ou charcoal) com um único acento sutil (dourado ou branco). Ícone abstrato e geométrico à esquerda, combinando um símbolo de código </> com um sinal de mais + (tecnologia + saúde). Tipografia limpa, sem serifa, peso leve a médio. Sem tagline. Fundo claro. Elegante, sofisticado, moderno.
```

## Opção 2 — Apenas tipografia (wordmark)

```
Crie um logotipo wordmark minimalista premium para "Codefy", empresa de tecnologia para clínicas. Apenas o nome "Codefy" em tipografia limpa e elegante, sem serifa, peso fino/light. Cores: preto ou charcoal escuro. Espaçamento generoso entre letras. Sem ícone, sem tagline, sem elementos extras. Design sofisticado, luxuoso, minimalista. Fundo claro. Ideal para marca de software B2B.
```

## Opção 3 — Monograma (ícone quadrado)

```
Crie um ícone de logotipo quadrado minimalista para "Codefy", empresa de software para saúde. Um monograma abstrato com a letra "C" formada por brackets de código < > entrelaçados. Linhas finas e elegantes, fundo escuro (charcoal #0d1117), traço branco ou dourado. Design premium, sofisticado, moderno. Sem texto. Formato 1:1, adequado para foto de perfil do Instagram.
```

## Opção 4 — Símbolo abstract geométrico

```
Crie um símbolo abstrato geométrico para a marca "Codefy" (tecnologia + saúde). Duas formas que se intersectam: um colchete angular < e uma cruz fina +, formando uma composição equilibrada e minimalista. Linhas limpas, sem preenchimento, apena traços pretos finos sobre fundo branco. Estilo premium, elegante, contemporâneo. Sem texto. Ideal para ser usado como ícone ao lado do nome.
```
